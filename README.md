# Alzheimer's Risk Prediction - Learning Project

Welcome! This is your step-by-step journey to build an AI model that predicts Alzheimer's risk using clinical data.

## 🎯 Project Goal
Build a machine learning model that can predict Alzheimer's risk from clinical data (age, cognitive test scores, health factors, etc.)

## 📚 Learning Path

### Phase 1: Python Fundamentals (Lessons 1-3)
- **Lesson 1**: Variables, Data Types, Lists, Dictionaries
- **Lesson 2**: Functions, Loops, Conditionals
- **Lesson 3**: File I/O (reading CSV files)

### Phase 2: NumPy Basics (Lessons 4-5)
- **Lesson 4**: Arrays, Shape, Basic Operations
- **Lesson 5**: Indexing, Slicing, Mathematical Operations

### Phase 3: Pandas Basics (Lessons 6-8)
- **Lesson 6**: DataFrames, Reading CSV
- **Lesson 7**: Data Cleaning, Handling Missing Values
- **Lesson 8**: Selecting Columns, Filtering Data

### Phase 4: Basic Math for AI (Lessons 9-10)
- **Lesson 9**: Vectors and Matrices (Intuition)
- **Lesson 10**: Basic Statistics (Mean, Median, Standard Deviation)

### Phase 5: Machine Learning Basics (Lessons 11-15)
- **Lesson 11**: Train-Test Split
- **Lesson 12**: Logistic Regression
- **Lesson 13**: Random Forest
- **Lesson 14**: SVM (Support Vector Machine)
- **Lesson 15**: Model Evaluation (Accuracy, Precision, Recall, AUC)

### Phase 6: Complete Project (Lessons 16+)
- **Lesson 16**: Full Pipeline
- **Lesson 17**: Feature Engineering
- **Lesson 18**: Model Comparison
- **Lesson 19**: Streamlit App (Optional)

## 🚀 Getting Started

1. **Install Python** (if not already installed)
   - Download from python.org (Python 3.8 or higher)

2. **Install required packages**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Start with Lesson 1**:
   ```bash
   python lessons/lesson_01_python_basics.py
   ```

## 💡 How to Use This Project

- **Work through lessons in order** - Each builds on the previous one
- **Run each lesson** - Don't just read, execute the code!
